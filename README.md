## Overview

Built the sample automation framework for this test. Imlemented 5 tests

## Technologies

* Java
* Page Design Pattern
* Page Factory Meth0d
* Selenium WebDriver
* TestNG
* Maven
* Chrome

## Executing the Test

* Run 'mvn clean install -DskipTests'
* Run 'mvn test'
